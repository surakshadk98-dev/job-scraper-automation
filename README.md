# Job Scraper Automation 🤖

## 📌 Problem Statement

Job searching manually on platforms like LinkedIn can be time-consuming and repetitive.

---

## 💡 Solution

This script automates job searching and extracts job listings based on keywords and location.

---

## 🚀 Features

* Search jobs by keyword
* Filter by location
* Extract job title, company, and location
* Save results to CSV

---

## 🛠 Tech Stack

* Python
* Selenium
* Pandas

---

## ▶️ How to Run

1. Install dependencies:
   pip install selenium pandas

2. Run the script:
   python job_scraper.py

---

## 📷 Output

Generates a `jobs.csv` file with job listings.

---

## ⚠️ Note

This project is for learning automation and web scraping concepts.
